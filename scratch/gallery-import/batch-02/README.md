# Gallery Batch 02 Intake

Use this folder only for staging and validating the next approved gallery photo batch.

## Source Originals

Place original Batch 02 source images in:

```text
scratch/gallery-import/batch-02/originals/
```

- Do not rename originals manually unless instructed.
- Keep originals unchanged.
- Preferred source image types are `.jpg`, `.jpeg`, and `.png` when needed.
- Avoid HEIC unless it will be converted during processing.
- Do not place edited or optimized images in `originals/`.

## Processed Files

`processed/` is reserved for generated import-ready copies.

Do not process, optimize, crop, or rename images until after source validation and manifest approval.

## Manifest Files

`manifest/` is reserved for proposed and final manifest JSON files.

No Supabase import should happen until after validation and manifest approval.
